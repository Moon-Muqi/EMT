# Excel页边距修改工具 - 安装说明

## 系统要求

### 最低配置
- **操作系统**: Windows 7 SP1 (64位)
- **处理器**: Intel Core i3 或同等性能
- **内存**: 2GB RAM
- **磁盘空间**: 100MB 可用空间
- **显示器**: 1024x768 分辨率

### 推荐配置
- **操作系统**: Windows 10/11 (64位)
- **处理器**: Intel Core i5 或更高
- **内存**: 4GB RAM 或更高
- **磁盘空间**: 500MB 可用空间
- **显示器**: 1920x1080 分辨率

## 安装步骤

### 方法一：绿色版安装（推荐）

1. **下载程序**
   - 从GitHub下载最新版本的发布包
   - 或从其他可信来源获取程序文件

2. **解压文件**
   - 将下载的压缩包解压到任意文件夹
   - 建议解压到 `C:\Program Files\Excel页边距修改工具\` 或桌面

3. **运行程序**
   - 双击 `Excel页边距修改工具.exe` 文件
   - 首次运行可能需要等待几秒钟加载

4. **创建快捷方式（可选）**
   - 右键点击exe文件
   - 选择"创建快捷方式"
   - 将快捷方式移动到桌面或开始菜单

### 方法二：系统集成安装

1. **以管理员身份运行**
   - 右键点击exe文件
   - 选择"以管理员身份运行"

2. **添加到系统PATH（可选）**
   - 将程序文件夹添加到系统环境变量PATH中
   - 这样可以在命令行中直接运行程序

## 首次运行

### 1. 启动程序
双击exe文件，程序会自动启动。

### 2. 等待初始化
首次运行可能需要：
- 下载必要的组件
- 初始化程序环境
- 创建配置文件

### 3. 界面显示
程序启动后会显示主界面，包含：
- 文件选择区域
- 参数设置区域
- 操作按钮
- 进度显示区域
- 日志显示区域

## 安全设置

### 杀毒软件设置
如果程序被杀毒软件误报：
1. 将程序文件夹添加到杀毒软件白名单
2. 或将exe文件添加到信任列表
3. 临时关闭实时保护（不推荐）

### Windows Defender设置
1. 打开Windows安全中心
2. 点击"病毒和威胁防护"
3. 在"病毒和威胁防护设置"中
4. 点击"添加或删除排除项"
5. 添加程序文件夹到排除列表

### 防火墙设置
通常不需要特殊设置，程序不会访问网络。

## 卸载程序

### 绿色版卸载
1. 直接删除程序文件夹
2. 删除桌面快捷方式（如果有）
3. 清理注册表项（可选）

### 清理残留文件
1. 删除配置文件（通常在程序文件夹中）
2. 清理临时文件
3. 删除日志文件（如果需要）

## 故障排除

### 程序无法启动
1. 检查系统要求是否满足
2. 确认文件完整性
3. 检查杀毒软件设置
4. 尝试以管理员身份运行

### 运行时错误
1. 检查系统权限
2. 确认磁盘空间充足
3. 检查内存使用情况
4. 重启计算机后重试

### 性能问题
1. 关闭其他占用资源的程序
2. 增加虚拟内存
3. 清理系统垃圾文件
4. 更新系统驱动
